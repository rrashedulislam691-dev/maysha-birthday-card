- 🎨 **Animated Title** - Dynamic text animations with bounce effects
- 💌 **Interactive Letter** - Click the button to reveal a personalized love letter
- 🎈 **Floating Balloons** - Smooth floating animations
- 🌟 **Sparkling Stars** - Twinkling decoration elements
- 🎭 **Circular Text** - Rotating text around a heart
- 📱 **Fully Responsive** - Works on all devices
- 💫 **Smooth Animations** - CSS animations and jQuery effects

## 🚀 Getting Started

### Option 1: Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/rrashedulislam691-dev/maysha-birthday-card.git
   cd maysha-birthday-card
   ```

2. Open `index.html` in your browser

### Option 2: GitHub Pages

1. Go to repository settings
2. Enable GitHub Pages from the `main` branch
3. Your site will be live at: `https://rrashedulislam691-dev.github.io/maysha-birthday-card`

## 🎨 Customization

### Change Names:
Edit the following in `index.html`:
- Button text: `Click Here Maysha`
- Greeting: `Dear Maysha`
- Card header: `To: Maysha 💖`
- Message content
- Birthday date: `27 May`

### Change Colors:
Edit `style.css` gradient and color values:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Update Images:
Place your images in the `images/` folder and update paths in HTML.
